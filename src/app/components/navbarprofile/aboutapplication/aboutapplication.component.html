AppTasks

Es una aplicación que permite al usuario registrar tareas, deberes, metas entre otros, con el propósito de que pueda recordar que tiene pendiente por realizar y así tener una vida más organizada.


Tecnologías usadas

AppTasks fue construida bajo las siguientes tecnologías:

  • Frontend:
    • Angular CLI 6
    • Bootstrap 4
  • API RESTful:
    • Node.js
    • Express.js
    • MongoDB
    • JWT
